# ContentHelper
自制的小工具。
## 功能
### Content Extractor
将导入的电子发票PDF文件中的发票号码和发票金额提取出来，再通过预设的快捷键按顺序依次粘贴到指定位置。默认使用快捷键Ctrl+Q将提取的内容按行粘贴，可自行更高快捷键键位。
### Excel Viewer
读取导入的.xlsx文件，并通过文字指令对行列进行搜索筛选和定位，以及录入数字。可配合Windows语音输入功能来输入文字指令。适合诸如在名单上指定人名后面录入数字的场景。

相关教程：https://github.com/blakezhang-tech/ContentHelper/blob/main/Tutorials/ExcelViewer/ReadMe.md
## 下载
在[Release页面](https://github.com/blakezhang-tech/ContentHelper/releases)寻找最近版本的ContentHelper.zip文件下载并解压，点击ContentHelper.exe文件启动该工具。
## 最低系统要求
操作系统：Windows 7 SP1 或更高版本（如Windows 10/11）

框架要求：[.NET 10.0 运行时](https://dotnet.microsoft.com/zh-cn/download/dotnet/10.0/runtime/desktop)

# Excel Viewer
## 主要功能
- Excel文件（*.xlsx）读取和另存。
- 根据关键词快速筛选定位指定记录。
- 通过指令文字操作数据记录的修改。
## 主要适用场景
- 需要大量对照名单录入数据的场合。例如：教师逐一翻动纸质试卷，将试卷成绩录入在_学生名单.xlsx_中。此时教师可使用语音输入法（例如Windows内置语音输入或搜狗输入法语音输入功能等）口述学生姓名，使语音输入法将姓名（支持同音但不同字）输入指令框。Excel Viewer据此在_学生名单.xlsx_中指定列表中检索，并显示结果，而后，教师再次利用语音输入法在指令框中输入相应成绩，Excel Viewer据此将成绩文字转换为阿拉伯数字填入指定单元格，从而实现对学生成绩的录入工作。

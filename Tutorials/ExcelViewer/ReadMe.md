# Excel Viewer
## 主要功能
- Excel文件（*.xlsx）读取和另存。
- 根据关键词快速筛选定位指定记录。
- 通过指令文字操作数据记录的修改。
## 主要适用场景
- 需要大量对照名单录入数据的场合。例如：教师逐一翻动纸质试卷，将试卷成绩录入在_学生名单.xlsx_中。此时教师可使用语音输入法（例如Windows内置语音输入或搜狗输入法语音输入功能等）口述学生姓名，使语音输入法将姓名（支持同音但不同字）输入指令框。Excel Viewer据此在_学生名单.xlsx_中指定列表中检索，并显示结果，而后，教师再次利用语音输入法在指令框中输入相应成绩，Excel Viewer据此将成绩文字转换为阿拉伯数字填入指定单元格，从而实现对学生成绩的录入工作。

### 数据搜索
此时，第1列默认作为**数据搜索列**（默认使用青色高亮显示），使用关键词搜索时，仅有第1列的数据作为搜索范围，其余各列的数据不会被关键词影响。

尝试在指令框中输入“10”，回车，则第1列（即名为【序号】的列）中符合以该关键词为开头的全部记录将被搜索到，并筛选显示在当前视图中：

![2026-01-18_182226](https://github.com/user-attachments/assets/d804359e-0539-4f95-860f-4199b7fa9132)

如果不想通过序号来搜索记录，而是想通过姓名来搜索，则可以将鼠标移动至【姓名】列的列头，点击右键，选择【将此列纳入搜索范围】，即可在随后的搜索活动中使该列也能够被搜索到：

![2026-01-18_182430](https://github.com/user-attachments/assets/3de8100f-353e-4d7d-baa6-8dfca519577a)

此时，由于【姓名】列也成为数据搜索列，故也被高亮显示：

![2026-01-18_182801](https://github.com/user-attachments/assets/d3771fe7-cdb7-4c6a-b28d-cf0e8cd7a200)

在指令框中尝试输入“王”，回车，则数据搜索列中所有以“王”为开头的记录都被筛选显示了出来：

### 录入数值
现在要录入成绩，在指令框右侧设置栏中勾选【数值改写】，从而启用通过指令框对指定单元格记录做数字录入。

![2026-01-18_183912](https://github.com/user-attachments/assets/d2cbeb92-43c0-4a3f-8dbe-1c4fb17c59f7)

鼠标点击某个单元格，在指令框中输入“100”、“一百”或“一百分”等，软件将其转换为阿拉伯数字并改写到相应的单元格中：

![2026-01-18_184121](https://github.com/user-attachments/assets/6a820bcc-3731-40cb-ae9b-1d88df65d3bf)

> [!NOTE]
>
> 在Excel Viewer的执行顺序中，搜索功能的优先级别高于数值改写。如果输入“100”、“一百”或“一百分”等文字作为指令，该指令将优先作为关键词执行搜索，只有在搜索不出任何结果时，才会进入数值改写环节。
>
> 因此，为了避免原本为了数值改写而输入的数字被当做关键词参与搜索，应将带有数字的列排除出数据搜索列的范畴。例如，在【序号】列的列头位置右键，选择【将此列移出搜索范围】：
